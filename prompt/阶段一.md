# 九、Electron 要求

Electron 必须采用：

```ts
contextIsolation: true
nodeIntegration: false
sandbox: true
```

本阶段应建立：

```text
electron/
├── main/
│   └── index.ts
├── preload/
│   └── index.ts
└── shared/
```

要求：

1. 主进程负责创建窗口；
2. Preload 只保留最小接口；
3. 渲染进程不能访问 Node.js；
4. 不得为了方便开启完整 Node 权限；
5. 不得建立任意文件读取接口；
6. 不得建立任意命令执行接口；
7. 不得实现后续 Agent IPC。
