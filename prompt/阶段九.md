- 已加载的目录缓存，切换目录后重新加载；
- 大目录加载时显示加载状态。

### 当前关联文件

- `src/components/FileTree.vue` — 文件树组件

## 9.2 检查现有错误处理

逐个检查以下模块的错误处理是否完善：

- **API 错误**：`electron/main/providers/deepseek-provider.ts` — SSE 流式请求的错误处理、HTTP 状态码处理、超时处理
- **文件错误**：`electron/main/tools/` 中各文件工具 — 文件不存在、权限不足、路径非法等
- **数据库错误**：`electron/main/database/` — 迁移失败、查询失败、连接失败
- **权限错误**：`electron/main/services/permission.service.ts` — 权限请求过期、重复决策
- **命令错误**：`electron/main/services/command.service.ts` — 命令被阻止、超时、进程崩溃

## 9.3 错误传递

- 主进程错误通过 IPC 返回 `{ ok: false, error: { type, message } }` 格式；
- Preload API 将错误透传到渲染进程；
- 前端 stores 捕获错误后更新 UI 状态；
- 前端组件展示错误提示（toast 或内联提示）。
